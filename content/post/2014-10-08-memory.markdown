## メモリ空き状況
```bash
free
```
free をみて，何の用途にも使っていない物理メモリ量を確認。  
```bash
vmstat -a
```
Inact をみて，キャッシュのうち，すぐ捨てられるページを確認。

## どんなプロセスが動いているか
```bash
top
```
または，
```bash
ps -aux
```
